Идея о проекте Jobi:
Jobi это платформа по нахождения работы для израильских подростков а так же для нахождения рабочей силы раздличных израильских бизнесов.
  У Jobi есть 3 вида платформы
    1. Сайт - сайт с аналитикой о работах, категории, рейтинг, коментарии и многое другое
    2. Телеграм канал - канал который выпускает карточки с лидирующими и новыми бизнесами на платформе
    3. Телеграм бот - бот который рабтает с ИИ и при помощи общения и узнавая пользователя дает ему самые подходящие вакансии

____________________________________________________

  Работа сайта:
    Сайт и есть цетр проекта, в нем можно смотреть вакансии, оценивать вакансии, создавать и тп
      1. index.html - главная страница где есть фильтр вакансий а так же показываются все работы которые есть
        Настройка фильтров:
          1. Город - например Беэр Шева
          2. Вид работы - физическая, разговорная 
          3. Возвраст - напиример от 14 до 16
          4. Рейтинг - например 5 звезд
          5. Язык - например русский и иврит
          6. Заработок - ~20 шекелей в час
          6. Популярные - Новые - Прибыльные
      2. profile.html - профиль пользователя в котором находится аналитика пользователя а так же базовые статистика
      3. messenger.html - Мессенджер для переписки с директором вакансии чтобы спрашивать вопросы перед тем как переходить в сам чат
      4. work.html - карточка вакансии
      5. business.html - страница для отправки новой вакансии для проверки
      6. admin.html - админ панель

____________________________________________________

  Работа телеграм канала:
    Телеграм канал нужен для быстрой публикации новых бизнесов и публикации лучших вакансий

____________________________________________________

  Работа телеграм бота:
    Телеграм бот работает по похожему алгоритму и помагает пользователю:
      1. ИИ общается с пользователем и спрашивает о том что умеет пользователь и хочет
      2. Анализирует данные пользователя
      3. Находит лучшие сервисы которые записаны в json файле и выдает карточки сервисов и если пользователю понравилось то он может перейти по ссылке на сайт
